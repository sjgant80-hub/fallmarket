Sovereign by design

Every listing runs offline, ships as one file when possible, and outlasts the vendor. No SaaS lock-in. No telemetry. If the internet goes dark, the tools still work.

Signed and verifiable

Listings, benchmarks, and transactions are all Ed25519-signed. Provenance travels with the fork tree. Trust is empirical, not marketed.

Agent-callable

Every SDK ships on npm. Every MCP works with Claude, Cursor, and Codex. Every API is Docker-ready. Machine-to-machine by default. The AI economy needs a canonical registry — this is it.
