About FallMarket

The sovereign registry of AI tools, agents, SDKs, MCP servers, and HTTP APIs.

What FallMarket is

FallMarket is the largest sovereign registry of AI tools on GitHub. As of July 2026, it lists 763 offerings across four kinds: SDKs on npm, MCP servers, HTTP APIs, and standalone one-file tools. Every listing is MIT-licensed, Ed25519-signed, and benchmarkable.

FallMarket is not a SaaS product. It is a static GitHub Pages site with a canonical JSON manifest, augmented by a Cloudflare Worker for search and benchmark verification. Zero user accounts. Zero telemetry. Every identity is an Ed25519 keypair. Every claim is verifiable.

Why sovereign matters

A sovereign tool runs on your machine, ships with its source, and does not lock you into a vendor. Every tool on FallMarket meets this bar. The single-file HTML tools open from file:// and work offline. The MCP servers run under any Claude, Cursor, or Codex client. The HTTP APIs ship with Dockerfile and docker-compose. The SDKs are pure ESM on npm with zero non-essential dependencies.

This matters because the AI agent ecosystem is at an inflection point. Buyers need to hire capabilities without signing up for a dozen different services. Sellers need to distribute their work without a middleman taking 30%. Agents need a canonical place to search, install, and hire from. FallMarket is that place, structured so nobody can rug-pull it — including the maintainers.

How it works

  1. Discover — search the catalog by keyword, kind (SDK / MCP / API / tool), or affinity vector.
  2. Verify — every listing carries a signed passport. Benchmarks are run by a neutral runner and signed independently.
  3. Install — copy the npm install, claude mcp add, or docker run command straight from the listing.
  4. Hire (Phase 2, coming soon) — pay in KCC or GBP · funds escrow · benchmark on delivery · release on pass.
  5. Fork — MIT license means you can improve any listing. Royalties propagate up the fork tree.

Who builds FallMarket

FallMarket is built and stewarded by AI-Native Solutions, the founding guild. AI-Native Solutions ships one-file HTML tools, MCP servers, SDKs, and HTTP APIs — the same shape as every FallMarket listing. Day-one inventory of 763 items comes from the guild's own estate.

Guild membership opens in Phase 3. Ship a benchmarked tool, get attested, and start earning royalties on every fork. Until then, all listings ship as MIT and the free tier is the default.
